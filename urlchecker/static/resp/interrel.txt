En django las tablas o relaciones son representadas por una clase con sus respectivos atributos(variables). Por defecto cada clase tendra un indice numérico como clave principal, a menos que se especifique en el constructor de una variable, que ésta será una clave principal.

Para realizar crear una relación entre dos tablas diferentes, se agrega una variable en la clase deseada con el constructor models.ForeignKey() y como argumento, el nombre de la relación a la cual se está referenciando. Si la relación entre las clases es con multiples tuplas se puede usar el constructor models.ManyToManyFields().
